# Perfil Pessoal — HTML e CSS

## Descrição
> Página de perfil pessoal desenvolvida como atividade da disciplina de Desenvolvimento Web Front End (DWFE). O projeto consiste em uma página web estática com informações pessoais, habilidades, hobbies e contato.

## Tecnologias utilizadas
> HTML5
> CSS3

## Estrutura da página
> - Apresentação com nome e foto qualquer;
> - Seção "Sobre mim";
> - Habilidades;
>  - Hobbies;
> - Contato com links para redes sociais.
